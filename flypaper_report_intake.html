Google Ads — Files to Pull
  • Campaign performance.csvCampaigns → All campaigns → Download → CSV. Date range = report period. Columns: Campaign, Impressions, Clicks, CTR, Avg CPC, Conversions, Conv. Rate.
  • Ad group performance.csvAd Groups → Download → CSV. Same date range. Columns: Campaign, Ad Group, Impressions, Clicks, CTR, Avg CPC, Conversions.
  • Conversions.csvGoals → Conversions → Download. Breakdown by conversion action.
  • Auction insights - search.csvCampaigns → Auction insights → Download. Shows competitor overlap.
  • Search terms.csvKeywords → Search terms → Download. Filter to report period.
GA4 / Website — Files to Pull
  • Traffic_acquisition_Session_source_medium.csvReports → Acquisition → Traffic acquisition. Set date range. Download as CSV. Primary dimension: Session source / medium.
  • Pages_and_screens_Page_path.csvReports → Engagement → Pages and screens. Download as CSV. Columns: Page path, Views, Users, Key events.
  • Queries_Organic_Google_Search_query.csvReports → Acquisition → Search Console → Queries (if linked). Download as CSV. Or pull from Search Console directly.
CallRail — Files to Pull
  • Call List-[date].xlsxCalls → All calls → Export. Set date range. Export as Excel. Includes individual call records with tags, duration, source, keyword.
  • Call attribution-[date].xlsxAttribution → Source report → Export. Shows call volume by source/medium with qualification rates.
The Trade Desk (Programmatic) — Files to Pull
  • Performance Insights (Kokai) XLSXReports → Performance Insights → Export. Set date range to full month. Export as Excel. This single file contains both Display and CTV — distinguished by the "Media Type" column (Video = CTV, Display = Display).
⚠️ VTC (view-through conversions) may not appear in this export. Note if VTC data needs to be pulled separately from the TTD dashboard.
Meta Ads — Files to Pull
  • Meta_campaigns_[date].csvAds Manager → Campaigns → Export table data → CSV. Date range = report period. Columns: Campaign name, Impressions, Reach, Clicks, CTR, CPM, CPC, Results, Cost per result.
  • Meta_adsets_[date].csvAd Sets tab → Export. Same columns. Include audience/targeting breakdown if available.
Microsoft / Bing Ads — Files to Pull
  • Bing_campaign_performance.csvReports → Performance → Campaign → Export. Date range = report period.
LSA / Google Business Profile — Files to Pull
  • LSA summary screenshot or CSV exportLocal Services Ads dashboard → Download report. Include leads, cost per lead, lead status breakdown.
  • GBP Insights exportGoogle Business Profile → Performance → Export (if available). Calls, direction requests, website clicks.
SEMrush / SEO — Files to Pull
  • Organic keywords exportSEMrush → Domain Overview or Organic Research → Export top ranking keywords. Include position, volume, traffic %, URL.
  • Position tracking exportIf position tracking is set up → Ranking Distribution → Export. Shows MoM position changes for tracked keywords.
